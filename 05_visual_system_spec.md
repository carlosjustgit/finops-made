# Visual System Spec

## Color Palette
Primary Blue: #1E4DFF
Dark Background: #0B1F3B
White Sections
Corporate greys for support elements

## Style Guidelines
- Enterprise minimalism
- High whitespace
- Strong typography hierarchy
- Dashboard-like UI for diagnostic tool
- No startup playful visuals

## Layout Rules
- Mobile-first
- Clear contrast on CTAs
- Executive tone
